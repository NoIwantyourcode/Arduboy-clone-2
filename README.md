# Arduboy-clone

This is a clone of a popular Arduino based, Gameboy style console known as the Arduboy!

# Why I made this

I have always wanted a game console, But my parents have never let me have one, Instead wanting to me to focus on my studies, So I decided to make this as a more clever rebellion (Don't worry, I asked them before designing this)
